Proposito del sitio

Este recurso educativo digital tiene como objetivo principal, mejorar las estrategias de aprendizaje, por medio de la enseñanza interactiva, comprendiendo Hojas de Estilo en Cascada CSS el cual es un lenguaje que define la apariencia de un documento escrito en un lenguaje de marcado (por ejemplo, HTML). y Frameworks CSS el cual es una biblioteca de estilos que puede ser usada para implementar diseños web. Aportan una serie de utilidades que pueden ser aprovechadas frecuentemente en los distintos diseños web. para posteriormente desarrollar su respectiva aplicación.

Hojas de Estilo en Cascada CSS

CSS es un estándar que se utiliza principalmente para separar las propiedades de la presentación (como el color de fondo, el tamaño de fuente o la sangría) del contenido. Aunque puede manejar muchos tipos de documentos, se utiliza principalmente como una forma de proporcionar información de presentación en páginas web.
La separación entre contenido y presentación aumenta la flexibilidad, lo que permite a los autores definir un conjunto de estilos que utilizarán miles de documentos, reduciendo así el tiempo y el trabajo que se debe invertir en cada actualización. Además, la implementación de CSS permite que el documento sea completamente semántico porque no requiere el uso de elementos de presentación.

Frameworks CSS

Un marco CSS es un programa que proporciona una variedad de opciones para que las utilice en el desarrollo de HTML, lo que aumenta la velocidad de desarrollo y facilita el proceso de creación de aplicaciones o sitios web. Parece que solo el uso de marcos CSS puede beneficiarse. El marco CSS le permite usar bibliotecas de código y tareas comunes para acelerar el desarrollo, por lo que no tiene que comenzar desde cero. Si trabaja con un equipo, todos tendrán el mismo código, por lo que el trabajo en equipo también se beneficia del marco CSS.
